## Estandarización de API REST con Java y Spring Boot

### Tecnologías usadas
* Java
* Spring Boot
* Maven
* JPA
* MySQL
* Swagger

### Dependencias necesarias
* Data JPA
* MySQL Connector
* Swagger 2
* Swagger UI
* Lombok
* Validation
* Mapstruct
